# Requirements Gathering

## Dataset
- Kaggle: www.kaggle.com/datasets/tobiasbueck/multilingual-customer-support-tickets/data 
- Size & Scope: ~30,000 labeled ticket entries featuring real-world customer IT support 

---

## Stakeholders
- Customers
- Support agents
- System administrators

---

## User Stories
- As a user, I want my ticket classified automatically
- As a user, I want a fast response
- As a system, I want to suggest solutions from past tickets

---

## Functional Requirements
- Classify tickets (priority/type/queue)
- Retrieve similar tickets
- Generate responses using RAG

---

## Non-Functional Requirements
- Fast response time (< 3 seconds)
- High accuracy classification
- Scalable retrieval system
- Reliable API integration
